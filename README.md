# Documentacion

mini proyecto de prueba de material-mkdocs.

Iniciar entorno virtual de python:

```bash
python -m venv venv 
# Activar entorno
source venv/bin/activate
# Verificar verion de pip
pip --version
# Instalar mkdocs-material
pip install mkdocs-material
# Abrir directorio en code
.code
```
