# Documentación

Mini proyecto de prueba de Material-MkDocs.

Iniciar entorno virtual de Python:

```bash
python -m venv venv
# Activar entorno
source venv/bin/activate
# Verificar versión de pip
pip --version
# Instalar mkdocs-material
pip install mkdocs-material
# Abrir directorio en Code
code .
```
